## Packages
framer-motion | Page transitions and scroll-triggered animations
react-hook-form | Form state management
@hookform/resolvers | Zod validation for forms
lucide-react | High-quality icons
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging tailwind classes

## Notes
- Tailwind CSS relies on CSS variables defined in index.css for the custom Bug Bully theme (Green, Yellow, Black).
- Static images (logo and flyers) are imported directly from @assets/.
- The contact form connects to POST /api/contact.
